
# Cook This

## Big Picture

A Sinatra app for storing and viewing recipes made by you and other users 

## Usage Instructions

`mkdir cook-this`

`cd cook-this`

`git clone git@gitlab.com:justindhall/cook-this.git`

`bundle install`

`bundle exec rake db:migrate`

`bundle exec shotgun`
